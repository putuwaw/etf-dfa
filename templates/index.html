Tentang Aplikasi

ETF-DFA adalah sebuah aplikasi berbasis website yang mengimplementasikan DFA (Deterministic Finite Automata) dengan ETF (Extended Transition Function), sehingga mampu untuk mengenali apakah sebuah string diterima atau tidak. Aplikasi ini dibuat untuk memenuhi tugas mata kuliah Teori Bahasa & Otomata. Pembuatan aplikasi ini menggunakan Python sebagai bahasa pemrograman utama, Flask sebagai web framewok, Bootstrap CSS, dan Heroku sebagai web hosting.
